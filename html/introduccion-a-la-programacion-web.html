Fundamentos de Programación Web

Introducción a la Programación Web

¿Qué es la Programación Web?

La Programación Web es una rama de la programación informática que permite desarrollar páginas y aplicaciones que funcionan a través de Internet o de una red local.

Mediante diferentes lenguajes de programación es posible crear sitios interactivos donde los usuarios pueden visualizar información, completar formularios, realizar compras, comunicarse o acceder a diversos servicios digitales.

En otras palabras, es el proceso de construir todo aquello que observamos cuando visitamos una página web.

  • Una página web es un documento digital que puede contener
    • Texto
    • Imágenes
    • Videos
    • Audio
    • Botones
    • Formularios
    • Enlaces
    • Animaciones
  • Estas páginas son visualizadas mediante un navegador como:
    • Google Chrome
    • Microsoft Edge
    • Mozilla Firefox
    • Safari

¿Qué es un sitio web?
Un sitio web es un conjunto de varias páginas relacionadas entre sí.

  • Ejemplos
    • Portal de una universidad.
    • Página oficial de una empresa.
    • Tienda virtual.
    • Plataforma educativa.
    • Blog personal.

¿Qué es una aplicación web?
Una aplicación web es un sistema mucho más avanzado que una página web.

Permite que el usuario interactúe constantemente con el sistema.

  • Ejemplos
    • Gmail
    • Google Drive
    • Moodle
    • Google Classroom
    • Facebook
    • WhatsApp Web
    • Canva

Arquitectura cliente-servidor.

¿Qué es la arquitectura Cliente-Servidor?
La arquitectura cliente-servidor es un modelo de comunicación en el que un dispositivo, denominado cliente, solicita información o servicios a otro dispositivo, denominado servidor, el cual procesa la solicitud y devuelve una respuesta.

Este modelo permite distribuir las funciones entre quienes consumen la información (clientes) y quienes la almacenan o administran (servidores).

¿Qué es un Cliente?
El cliente es el dispositivo o programa que solicita un servicio al servidor.

Generalmente el cliente es un navegador web instalado en una computadora, teléfono móvil o tableta.

  • Ejemplos de clientes
    • Google Chrome
    • Microsoft Edge
    • Mozilla Firefox
    • Safari
    • Opera

También pueden actuar como clientes aplicaciones como:

    • WhatsApp Web
    • Spotify
    • Microsoft Teams
    • Google Classroom

¿Qué es un Servidor?
El servidor es una computadora o sistema especializado que almacena información, procesa solicitudes y responde a las peticiones realizadas por los clientes.

Los servidores funcionan las 24 horas del día para atender miles o millones de usuarios de forma simultánea.

  • Entre sus funciones principales se encuentran:
    • Almacenar páginas web.
    • Gestionar bases de datos.
    • Procesar información.
    • Ejecutar aplicaciones.
    • Controlar el acceso de los usuarios.
    • Enviar archivos al navegador.

Tipos de sitios web

Tipos de Sitios Web


Los sitios web se pueden clasificar según el objetivo para el cual fueron creados, el público al que están dirigidos y las funciones que ofrecen. Cada tipo de sitio web responde a necesidades específicas de información, comunicación, comercio, educación o entretenimiento.

1. Sitios Web Informativos
Los sitios web informativos tienen como finalidad principal ofrecer información clara y organizada sobre un tema, una institución, una comunidad o un servicio. Su objetivo no es necesariamente vender productos, sino mantener informados a los visitantes.

Estos sitios suelen ser utilizados por organismos gubernamentales, universidades, centros educativos, organizaciones sin fines de lucro y empresas que desean compartir información relevante con el público.

  • Características:
    • Información estructurada y fácil de consultar.
    • Contenido actualizado periódicamente.
    • Navegación sencilla.
    • Enfoque educativo o informativo.
  • Ejemplos:
    • Portal del Ministerio de Educación.
    • Sitios de ayuntamientos.
    • Enciclopedias en línea como Wikipedia.
  • Ventajas:
    • Facilitan el acceso al conocimiento.
    • Mejoran la comunicación institucional.
    • Sirven como fuente de consulta permanente.

2. Blogs

Un blog es un tipo de sitio web donde una persona o grupo publica contenidos llamados "entradas" o "artículos". Estas publicaciones aparecen generalmente en orden cronológico, mostrando primero las más recientes.

Originalmente se utilizaron como diarios personales, pero actualmente son herramientas ampliamente utilizadas para compartir conocimientos, experiencias y opiniones.

  • Características:
    • Publicaciones periódicas.
    • Organización cronológica.
    • Posibilidad de recibir comentarios.
    • Contenido especializado o personal.
  • Ejemplos:
    • Blogs educativos.
    • Blogs de tecnología.
    • Blogs de viajes y turismo.
  • Importancia:
    Los blogs permiten difundir información de manera rápida y favorecen la interacción entre el autor y los lectores.


3. Sitios Web Corporativos o Empresariales
Son sitios desarrollados para representar la imagen digital de una empresa, institución o negocio. Funcionan como una carta de presentación en Internet y permiten a los clientes conocer los productos, servicios y valores de la organización.

  • Características:
    • Información institucional.
    • Presentación de productos o servicios.
    • Datos de contacto.
    • Formularios para atención al cliente.
  • Secciones comunes:
    • Inicio.
    • Sobre Nosotros.
    • Servicios.
    • Contacto.
    • Noticias o novedades.
  • Ejemplos:
    • Microsoft.
    • Samsung.
    • Bancos y empresas de telecomunicaciones.
  • Beneficios:
    • Mejora la presencia digital.
    • Incrementa la credibilidad.
    • Facilita la comunicación con clientes y usuarios.


4. Sitios de Comercio Electrónico (E-Commerce)

Los sitios de comercio electrónico permiten realizar transacciones comerciales a través de Internet. Los usuarios pueden buscar productos, agregarlos a un carrito de compras y completar el pago desde cualquier lugar.

El crecimiento del comercio electrónico ha transformado la forma en que las personas compran y venden productos y servicios.

  • Características:
    • Catálogo digital de productos.
    • Carrito de compras.
    • Pasarelas de pago.
    • Gestión de pedidos y envíos.
  • Ejemplos:
    • Amazon.
    • Mercado Libre.
    • eBay.
  • Ventajas:
    • Disponibilidad las 24 horas.
    • Alcance global.
    • Comodidad para compradores y vendedores.

5. Portales Web
Un portal web es un sitio que reúne múltiples recursos, servicios e información en una sola plataforma. Actúa como un punto de acceso a diferentes herramientas y contenidos.

Los portales suelen ser utilizados por instituciones educativas, empresas y organizaciones gubernamentales.

  • Características:
    • Integración de diversos servicios.
    • Acceso centralizado.
    • Personalización del contenido.
    • Múltiples secciones informativas.
  • Ejemplos:
    • Portal Educativo del MINERD.
    • Yahoo.
    • Portales universitarios.
  • Beneficios:
    • Facilitan la búsqueda de información.
    • Ahorran tiempo al usuario.
    • Centralizan recursos importantes.

Herramientas necesarias para el desarrollo web.

1. Editor de Código o IDE
Un editor de código es una aplicación que permite escribir y editar el código fuente de una página web. Es una de las herramientas más importantes para cualquier desarrollador.

  • Funciones principales:
    • Resaltado de sintaxis.
    • Autocompletado de código.
    • Detección de errores.
    • Organización de archivos.
  • Ejemplos:
    • Visual Studio Code (VS Code).
    • Sublime Text.
    • Notepad++.
    • WebStorm.
  • Importancia:
    Los IDE Facilitan la escritura de código, aumenta la productividad y ayuda a detectar errores antes de ejecutar el proyecto.

Los navegadores permiten visualizar y probar las páginas web desarrolladas.

    • Mozilla Firefox.
    • Microsoft Edge.
    • Safari.

    Los navegadores nos ayudan verificar cómo se comporta el sitio web desde la perspectiva del usuario final.

4. Herramientas para Diseño (UI/UX)
Estas herramientas ayudan a diseñar la apariencia y experiencia de usuario antes de comenzar la programación.

  • Importancia:
    Permiten visualizar el proyecto antes de desarrollarlo, reduciendo errores y mejorando la experiencia del usuario.
