Fundamentos de Programación Web

Portada

Información General

Este curso tiene como propósito desarrollar las competencias fundamentales en Programación Web mediante una metodología de educación híbrida, que combina clases presenciales con actividades virtuales. Los participantes aprenderán a crear páginas web utilizando HTML, CSS y JavaScript, mientras fortalecen habilidades como el aprendizaje autónomo, la resolución de problemas, el trabajo colaborativo y el uso de herramientas digitales. A lo largo del curso realizarán actividades prácticas, foros de discusión, evaluaciones en línea y un proyecto final aplicado a una necesidad real de su comunidad educativa.

Objetivo general del curso

Desarrollar en los estudiantes las competencias fundamentales de Programación Web mediante una metodología de educación híbrida, integrando clases presenciales y actividades virtuales para el aprendizaje de HTML, CSS y JavaScript, promoviendo el aprendizaje autónomo, el trabajo colaborativo y el desarrollo de proyectos web aplicados a necesidades reales.
