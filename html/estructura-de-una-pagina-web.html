Introducción a HTML5.

¿Qué es HTML5?

HTML5 es la versión más reciente del lenguaje de marcado HTML (HyperText Markup Language), utilizado para crear y estructurar páginas web. Este lenguaje permite organizar el contenido de un sitio web mediante elementos como títulos, párrafos, imágenes, enlaces, tablas y formularios.

HTML5 fue desarrollado para mejorar las capacidades de las versiones anteriores, incorporando nuevas etiquetas y funcionalidades que facilitan la creación de sitios web modernos, interactivos y compatibles con diferentes dispositivos, como computadoras, tabletas y teléfonos inteligentes.

Importancia de HTML5 en el Desarrollo Web
HTML5 es considerado la base de cualquier página web. Sin HTML, los navegadores no podrían interpretar ni mostrar correctamente la información que encontramos en Internet.

Algunas ventajas de HTML5 son:

  • Mayor compatibilidad con navegadores modernos.
  • Mejor organización y estructura del contenido.
  • Integración nativa de audio y video sin necesidad de complementos externos.
  • Mejor soporte para aplicaciones web dinámicas.
  • Optimización para dispositivos móviles.
  • Favorece el posicionamiento en motores de búsqueda (SEO).

Estructura Básica de un Documento HTML5

Todo sitio web comienza con un documento HTML. Para que un navegador pueda interpretar y mostrar correctamente una página web, es necesario seguir una estructura básica que organiza el contenido y proporciona información sobre el documento.

La estructura de un documento HTML funciona como el esqueleto de una página web, definiendo dónde se colocarán los textos, imágenes, enlaces, tablas, formularios y otros elementos visuales.

<!DOCTYPE html>
Indica al navegador que el documento utiliza HTML5.

<html>
Es el elemento principal que contiene toda la página.

<head>
Incluye información sobre el documento, como el título, configuración de idioma y metadatos.

<title>
Define el nombre que aparecerá en la pestaña del navegador.

<body>
Contiene todo el contenido visible para los usuarios.

<h1>
Representa un encabezado principal.

<p>
Representa un párrafo de texto.

Aplicaciones de HTML5


HTML5 se utiliza para desarrollar:
  • Sitios web corporativos.
  • Blogs y portales informativos.
  • Tiendas en línea.
  • Plataformas educativas.
  • Aplicaciones web.
  • Sistemas de gestión empresarial.
  • Redes sociales.

Prácticamente cualquier sitio web moderno se construye utilizando HTML5 como base estructural.

La estructura básica de un documento HTML está compuesta por las etiquetas <!DOCTYPE html>, <html>, <head> y <body>. Cada una cumple una función específica que permite organizar correctamente la información de una página web. Comprender esta estructura es fundamental para cualquier estudiante de desarrollo web, ya que constituye la base sobre la que se construyen todos los sitios y aplicaciones web modernas.

Etiquetas y Elementos en HTML

Etiquetas y Elementos en HTML

Las etiquetas y los elementos son los componentes fundamentales de HTML. Gracias a ellos, los desarrolladores pueden estructurar y organizar el contenido de una página web, indicando al navegador cómo debe interpretar y mostrar la información.

Comprender la diferencia entre una etiqueta y un elemento es esencial para comenzar a desarrollar sitios web de manera correcta y eficiente.

¿Qué es una Etiqueta HTML?

Una etiqueta (tag) es una instrucción escrita entre los símbolos de menor y mayor que (< >) que indica al navegador cómo tratar una parte específica del contenido.

Las etiquetas generalmente aparecen en pares:

  • Etiqueta de apertura.
  • Etiqueta de cierre.

Ejemplo: <p>Este es un párrafo.</p>

<p> es la etiqueta de apertura.
</p> es la etiqueta de cierre.

La barra inclinada (/) indica el final de la etiqueta.

¿Qué son los elementos?

Un elemento es el conjunto completo formado por:
  • La etiqueta de apertura.
  • El contenido.
  • La etiqueta de cierre.

Listas

Las listas sirven para organizar información de manera ordenada o desordenada, facilitando la lectura.
Son muy útiles cuando queremos mostrar varios elementos relacionados.

<ul>
       <li>Manzana</li>
       <li>Pera</li>
       <li>Naranja</li>
</ul>


Tablas

Las tablas permiten organizar datos en filas y columnas para compararlos o presentarlos de forma clara.

<table border="1">
      <tr>
            <th>Nombre</th>
            <th>Edad</th>
      </tr>
      <tr>
           <td>Juan</td>
           <td>18</td>
       </tr>
</table>

ste código mostrará una tabla con el nombre y la edad de una persona.
